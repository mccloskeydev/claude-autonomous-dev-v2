# Plan

Generate implementation approaches and select the best one.

## Arguments
$ARGUMENTS - The task/feature to plan

## Instructions

Activate the `plan-alternatives` skill:

1. Understand the requirements
2. Research if needed (use researcher subagent)
3. Generate at least 3 distinct approaches
4. Evaluate each with pros, cons, risks
5. Create comparison matrix
6. Select best approach with rationale

Save output to `specs/plan.md`

# Resume

Resume development from the most recent checkpoint.

## Instructions

1. Find most recent checkpoint file in specs/checkpoint-*.md
2. Read the checkpoint document
3. Summarize:
   - What was being worked on
   - Progress so far
   - Outstanding items
   - What to do next
4. Continue from "What to Do Next" section

If no checkpoint exists:
- Read specs/features.json for feature status
- Read specs/progress.md for recent activity
- Continue with next incomplete feature

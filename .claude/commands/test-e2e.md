# Test E2E

Run end-to-end browser tests using Puppeteer MCP.

## Arguments
$ARGUMENTS - Feature to test, or "all" for full E2E suite

## Instructions

Activate the `test-e2e` skill:

1. Ensure dev server is running
2. Define test scenarios for the feature
3. Execute tests using Puppeteer MCP tools:
   - Navigate to pages
   - Fill forms
   - Click buttons
   - Take screenshots
   - Evaluate page state
4. Document results in specs/e2e-results.md
5. Log any failures to specs/bugs.md

# Status

Show current development status.

## Instructions

Read and summarize:

1. **Features** from specs/features.json
   - Count: complete vs incomplete
   - List incomplete features

2. **Progress** from specs/progress.md
   - Recent activity
   - Current iteration count

3. **Bugs** from specs/bugs.md
   - Count: fixed vs unfixed
   - List unfixed bugs

4. **Git status**
   - Uncommitted changes
   - Recent commits

Output a concise status report.

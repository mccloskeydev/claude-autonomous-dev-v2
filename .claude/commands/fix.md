# Fix

Systematically investigate and fix a bug.

## Arguments
$ARGUMENTS - Bug description or ID from specs/bugs.md

## Instructions

Activate the `evaluate-fix` skill:

1. Understand the bug (symptoms, expected vs actual)
2. Reproduce the bug
3. Identify root cause
4. Write regression test (must fail first)
5. Fix the code
6. Verify test passes
7. Run full test suite
8. Commit the fix
9. Update specs/bugs.md with FIXED status
